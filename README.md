# AshOptimized

MIUI12.5优化模块,调度偏流畅省电,CPU调度代码少甚至没有,减少与部分调度冲突
仅在Mi10Utral/OnePlus7Pro机型测试过,理论上全机型通用,请自己自行测试

## 版本号

v1.0.6(更新日期: 2021/11/20)

## 更新内容
- 新增适配机型OnePlus7Pro
- 新增判断机型并输出信息
- 新增部分判定选择安装
- 更新DozeLog输出,新增电量显示
- 新增功能开关选项
- 更新Config.conf配置文件(需更新Config.conf配置文件)
