# AshOptimized

MIUI12.5优化模块,调度偏流畅省电,CPU调度代码少甚至没有,减少与部分调度冲突
仅在Mi10Utral机型测试过,理论上全机型通用,请自己自行测试

## 版本号

v1.0.4(更新日期: 2021/10/24)

- 移除AshScripts中的PowerWhite脚本
- 新增PowerWhite Crond脚本
- 修复部分脚本运行判定错误的问题
- 更新config配置(重启前需要用户手动删除config配置)
